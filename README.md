# Mobile-Price-Prediction
Now a day’s Mobiles become part and parcel human life. And demand for mobiles is very high. And the price of mobile phone depends on its features. As number of features and their quality increases, price of mobile increases and vice versa. Now fixing price for a mobile based on its features is a big deal for companies to keep their competition alive. And price prediction can be done based on mobile features and previous mobiles data. In the area of decision making based on past data Machine Learning plays a vital role. This is a web application which collects inputs from user using HTML forms and predicts the best price for mobile using Random Forest Algorithm.

# Random Forest Algorithm :

Step-1: In Random forest n number of random records are taken from the data set having k number of records.

Step-2: Individual decision trees are constructed for each sample.

Step-3: Each decision tree will generate an output.

Step-4: Final output is considered based on Majority Voting or Averaging For Classification and regression respectively.
